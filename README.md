# دور / Turn

منصة حجوزات وقوائم انتظار للمطاعم — عربية RTL بالكامل.

**التقنيات:** Next.js 15 · TypeScript · Tailwind CSS v4 · Supabase · خطوط Almarai/Tajawal

## البدء

```bash
cp .env.example .env.local   # ثم املأ قيم Supabase
npm install
npm run dev
```

افتح [http://localhost:3000](http://localhost:3000).

راجع `CLAUDE.md` لتفاصيل البنية وقاعدة البيانات.
